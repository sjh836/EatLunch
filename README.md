## 1. 프로젝트 개요
Jswing(javax.swing.*)을 이용하여 자바 어플리케이션을 만들었다. 주제는 우리 학교 근처 식당들의 데이터(약 300건)들을 넣고 조건에 맞게 정보를 유저에게 제공해준다.

주요기능과 동작화면, 스크린샷은 아래 PPT에 있다.

**PPT링크:** https://www.slideshare.net/JungHoonLee41/syu-75539907
